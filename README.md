# 가족 용돈/재테크 프로토타입

브라우저에서 바로 실행 가능한 프로토타입입니다.

## 실행 방법
1. `python3 -m http.server 8000`
2. 브라우저에서 `http://localhost:8000/app/` 접속

## 현재 구현 범위
- 기본 멤버: 나/첫째/둘째
- 상단 멤버 탭(전체/개별) 전환
- 개별 탭에서 대상 멤버 자동 고정 입력
- 거래 입력(용돈 수입/저축/투자/지출)
- 카테고리 입력(예금/적금/주식/ETF/코인 등)
- 대시보드 KPI(수입/저축/투자/지출, 비율, 평균 수익률)
- 월 예산/저축 목표(멤버별)
- 검색 필터(기간/종류/카테고리)
- 일자별 용돈 합계 / 카테고리별 합계 / 거래내역
- 데이터 영속 저장(LocalStorage)
